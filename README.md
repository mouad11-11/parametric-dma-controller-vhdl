# parametric-dma-controller-vhdl
Fully synthesizable, parametric Single-Channel DMA Controller in VHDL-2008. Features a robust burst-transfer FSM, integrated synchronous FIFO buffering, and a self-checking verification testbench implemented using OSVVM Scoreboards.

for easier testing copy the content of setup.do and paste it into modelsim command line
